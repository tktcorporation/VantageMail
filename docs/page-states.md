# VantageMail ページ状態マップ

## 状態駆動モデル

URL ルーティングなし。以下の3つの状態変数で全画面が決まる:

| 状態変数 | 型 | 値 |
|----------|------|------|
| `activeCategory` | `"all" \| "people" \| "notifications" \| "newsletters"` | Smart Inbox のフィルタ |
| `selectedThreadId` | `string \| null` | 選択中のスレッド |
| `showSettings` | `boolean` | 設定画面の表示 |

## コンポーネント依存マップ

```
App
├── StoreContext.Provider (Zustand)
│   └── InnerAppShell
│       ├── useSync()                    ← Gmail API 同期
│       ├── useKeyboardShortcuts()       ← グローバルキーボード
│       │
│       ├── AppLayout (3カラム CSS Grid)
│       │   ├── [slot: sidebar]
│       │   │   └── Sidebar
│       │   │       ├── Smart Inbox Categories ← activeCategory 制御
│       │   │       ├── Account Selector       ← activeAccountId 制御
│       │   │       └── Settings/Add buttons   ← showSettings 制御
│       │   │
│       │   ├── [slot: list]
│       │   │   └── ThreadList
│       │   │       ├── SearchBar              ← 検索UI（API未接続）
│       │   │       ├── CategoryCard[]         ← all時のみ
│       │   │       └── ThreadItem[]           ← フラット or カード内
│       │   │
│       │   └── [slot: main]
│       │       ├── ThreadView (showSettings=false)
│       │       │   ├── Thread Header
│       │       │   ├── MessageItem[]          ← useThreadMessages()
│       │       │   └── Reply Bar
│       │       └── AccountSettings (showSettings=true)
│       │           ├── Account Card[]
│       │           └── Add Account Button
│       │
│       └── CommandPalette (モーダル、常時マウント)
```

## キーボードショートカット一覧

| キー | アクション | 対象 |
|------|-----------|------|
| `J` / `↓` | 次のスレッド | ThreadList |
| `K` / `↑` | 前のスレッド | ThreadList |
| `E` | アーカイブ | 選択中スレッド |
| `S` | スター切替 | 選択中スレッド |
| `#` | ゴミ箱 | 選択中スレッド |
| `C` | 新規作成 | （未接続） |
| `R` | 返信 | （未接続） |
| `/` | 検索フォーカス | SearchBar |
| `Cmd+K` | コマンドパレット | グローバル |
